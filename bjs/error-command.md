By default we have such error message:

<img src="../.gitbook/assets/image (91) (1) (1).png" alt="" data-size="original">

![](<../.gitbook/assets/image (93) (1).png>)

{% hint style="success" %}
You also can use BJS in error command
{% endhint %}
